# AGENTS 指南

本项目允许并鼓励代理在执行任务时读取 `docs/` 目录下的文档作为主要上下文来源。

## 文档读取规则

- 代理可以主动读取 `docs/` 下的 `.md` 文件获取背景信息、协议说明和配置说明。
- 当用户问题与环境、协议、奖励、观测、配置相关时，应优先参考 `docs/` 内容再进行修改或回答。
- 若 `docs/` 与代码实现存在冲突，应以当前代码为准，并在回复中明确指出差异。
- 不要修改 `docs/` 文档原意；如需整理格式，可仅做结构化优化（标题、列表、表格、代码块）。

## 建议阅读顺序

1. `docs/introduction.md`
2. `docs/env.md`
3. `docs/protocol.md`
4. `docs/agent_description.md`
